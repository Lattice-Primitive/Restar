# 板级支持包（BSP）部分说明
**from ZhiyuanMao(2019th)**

## 前言
这个库从2021年9月开始写，当时写这个库的契机一开始是为了准备当年（2022）省赛对抗，打算使用HAL库来移植学长们之前留下，基于标准库的驱动库。后来有了电赛，我就开始着手对电赛的准备，电赛所使用的开发板基于TI公司的MSP432系列，考虑到了跨平台需求，于是接下来的驱动库，我都将它们支持了跨平台。
目前这个驱动库中一共有十余个库，也有不少因为没时间写的库，这就留给之后的人来补充了。

## 说明
1. 这个库遵循[EEPS-alpha](../Manuals/Elite_Embeded_Programing_Std.md),后来人在迭代过程中也务必遵守，也可对其做出合理改进。
2. 若要进行二次开发，请务必仔细读完上面提供的规定，并最好有编写项目代码的经验。
3. 接下来的开发可以着眼于对我没写完的库进行开发，对我写完的库进行迭代，或是自己创建新的驱动库
4. 记得写个示例，放在example中
