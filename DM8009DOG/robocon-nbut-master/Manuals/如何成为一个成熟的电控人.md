# 如何成为一个成熟的电控人
### 单片机(嵌入式)开发入门学习路线
**from Zhiyuan Mao**

## 前言
对于刚进入实验室的同学来说，可能也听说了，我们实验室的三大路线，机械、电控、视觉AI，这三个方向我都有所涉足(其余两边涉了但只涉了一点点)，我是电类专业出身，大二的时候正式选择了电控这个方向，其实是有些慢了，希望你们要更早确定自己的爱好方向。

电控，顾名思义就是电子控制。电控人的工作主要是让我们的目标系统(主要是芯片、开发板)按照我们预先规划运行并达到一定目的，顺便再对系统整体的电路也需要大概的规划。与电相关的东西，都是由电控人负责。电控人需要有强大的C语言编程能力和一定的电路知识。

如果你对电控感兴趣，选择了电控，那接下来可以看看，如何成为一个成熟的电控人。

## 目标
我认为，成为一个成熟的电控人不难，只要好好学习天天向上。大概只需要1到1.5年不等。

一个成熟的电控人应该达到的标准：

1. 可以很**熟练**地使用C语言进行开发，并能熟练使用C语言的一些特性。
2. 可以看着官方的库和参考手册对单片机直接进行开发，不明白的地方擅长利用多种媒介**自主**查询资料。
3. 可以独立通过多种调试方法配合参考手册**自主**排查出问题。
4. 可以按照自己的需求**自主**开发一些库。
5. 有大型比赛的**主力经验**，这很重要,作为主力参加了比赛之后

达到上述的目标，你就可以轻松胜任一些单片机系统的开发任务了。随着技术的成长，你也可以选择其他的道路，因为低端嵌入式单片机的工资是不高的，不如嵌入式Linux，FPGA，ZYNQ这些。不过你已经成熟了，需要自己选择继续学习的方向。

## 关于学习路线
学习路线主要还是看个人，像我的学习路线就特别坎坷，啥都学过，啥都不精，这是我的学习路线：
1. Python (暑假里无聊学的)
2. C (学了Python之后难度不大) 
3. Arduino (易学易入门)
4. SolidWorks (所以机械的活我也会一点)
5. 8051 (最经典的单片机架构)
6. ARM (目前嵌入式的主流架构)
7. 嵌入式Linux (~~难学的要死，淦~~)
8. ...

这里列出了我的主要学习路线，里面还有很多细节没有说明
